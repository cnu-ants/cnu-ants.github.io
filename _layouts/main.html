{{ site.labname }} research group in Chungnam National University focuses on programming languages, program analysis and testing, program verification, and security. The research topics include, but are not limited to:

  • Program Analysis: Devise novel analysis techniques to extract valuable semantic properties as well as detect bugs and security vulnerabilities from real-world programs.
  • Testing: Auto-generate test cases efficient to expose faults from programs, and localize root causes of the faults.
  • Binaries: Analyze binary executables to detect hidden malicious behaviors in them, and to lift them to a more understandable format.
  • Mobile applications and ML code : Analyze mobile applications and ML code to detect bugs and privacy issues, which make them safe and dependable.

We are recruiting motivated undergraduate, graduate students and postdoctoral researchers! If you are interested in our research and laboratory, please contact Sungho Lee.
